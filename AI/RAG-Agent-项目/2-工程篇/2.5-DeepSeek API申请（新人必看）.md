DeepSeek API 的申请非常简单，直接到： [<u>https://platform.deepseek.com/api_keys </u>](https://platform.deepseek.com/api_keys)然后创建 api key 就行了，复制好 key 填到 `application.yml` 中就可以了。

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501223649469.png)

记得充钱。

然后 URL 为 [<u>https://api.deepseek.com/v1 </u>](https://api.deepseek.com/v1)model 为 `deepseek-chat`

key 就填你刚刚申请的就好了。

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501223652928.png)

启动项目后，可以通过聊天助手这里测试，随便输入内容，如果有内容返回，就表明配置正确了。

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501223658054.png)
