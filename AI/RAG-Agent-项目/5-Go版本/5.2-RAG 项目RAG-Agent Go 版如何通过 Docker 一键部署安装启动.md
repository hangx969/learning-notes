RAG 项目RAG-Agent Go 版 AI 知识库代码库已经完成了，为了降低上手的难度，这里提供了 <u>Go</u> 版 Docker 的一键部署方案，基本上一天就能搞定 MinIO、ElasticSearch、Redis、MySQL、Kafka 的前置环境安装。

手把手教大家！

## 一、申请向量化 API key

登录阿里百炼： [<u>https://bailian.console.aliyun.com/?tab=model#/api-key </u>](https://bailian.console.aliyun.com/?tab=model#/api-key)，申请 API key，然后复制，随后需要粘贴到 config.yaml 中：

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224415979.png)

在这里可以查看： [<u>https://bailian.console.aliyun.com/?tab=model#/model-market/all </u>](https://bailian.console.aliyun.com/?tab=model#/model-market/all)通用文本向量-v4

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224418702.png)

会免费送一批额度，基本上够我们RAG-Agent使用了，😄 等用完了再充值。

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224421819.png)

OK，然后把申请的 API key 粘贴到 config.yaml 中。注意这里我们用的是 2048 维，一定要和 ElasticSearch 匹配哦。

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224425147.png)

## 二、申请 DeepSeek API key

我们这里的模型用的是 DeepSeek。

DeepSeek API 的申请非常简单，直接到： [<u>https://platform.deepseek.com/api_keys </u>](https://platform.deepseek.com/api_keys)然后创建 api key 就行了。

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224428170.png)

记得充钱。

复制好 key 填到 `config.yaml` 中就可以了。然后 URL 为 [<u>https://api.deepseek.com/v1 </u>](https://api.deepseek.com/v1)model 为 `deepseek-chat` ，key 就填你刚刚申请的就好了。

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224431216.png)

## 五、启动项目

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224522771.png)

如果你是新手，出现了飙红，不要慌。

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224526227.png)

在 goland 中启动终端执行 `go mod tidy` 让依赖重新加载一下。

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224529295.png)

**这里提醒两点，如果你本地已经安装了 MySQL 和 Redis，也不想在 Docker 中拉取，那么就可以在配置文件这里直接改成你本地的密码&#x20;**。

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224532598.png)

比如说 MySQL 的密码就改成你本地的，Redis 的也改成你本地的，如果你想使用 Docker 拉取的镜像，就可以保持现状。

接着新打开一个终端，记得仍然要在项目的根目录，执行： `go run cmd/server/main.go`

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224535268.png)

假如以上这些问题你都没有出现，那么恭喜你 🎉， **RAG-Agent Go 版的后端你已经启动成功了。**

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224539031.png)

来，我们在终端切到前端目录，在 paismart-go/frontend 目录下，先执行 `pnpm i` install 一下前端。

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224542507.png)

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224545583.png)

然后执行 `pnpm run dev` 就可以启动前端了。

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224549169.png)

![](https://raw.githubusercontent.com/hangx969/upload-images-md/main/20260501224552147.png)
