# 什么是接口interface
以手机举例，给手机的充电口定义了一个标准（type c），只要充电线厂商兼容了这个标准，厂商的线就能用来给手机充电。

以开发app举例，我开发了一个app，不同的客户用，他们用的数据库可能不同：mysql、sql server、postgresql等。我在app中定义一个接口，让每个数据库都适配这个接口，就能无缝的适配各种数据库了。

Go语言的接口是一种类型Type，定义了一组方法的集合，但是接口不需要去实现他们，这些方法可以被不同的类型实现，进而就是这个类型实现了这个接口。

接口被广泛应用于许多标准库和框架中，通过接口可以使不同类型之间实现相同的行为，从而达到代码复用和扩展的目的，实现不同类型之间无缝切换。
