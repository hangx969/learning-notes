# Fiddler

> [!summary] ==Fiddler Classic== is an HTTP debugging proxy for capturing and analyzing network traffic.

---

## 安装与配置

- 安装选择：==Fiddler - Classic==

- 安装后的配置：
  - HTTPS
  - Trust root cert

## Filter

> [!tip] 选择特定的进程的 trace 来过滤无关流量

## 快捷键

| 快捷键 | 功能 |
| --- | --- |
| `Ctrl+F` | 查找 |
| `Ctrl+1` | 高亮 |
| `D` | 找到 URL 一样的条目 |
| `P` | 找到 parent |
| `C` | 找到 client |
