## 科普视频制作的工具链有多长

传统的科普视频制作链路：

**写稿 → 做PPT → 调动画 → 录屏 → 剪辑**

每一步都需要不同的工具和技能。PPT动画不是随便拖几个元素就能出效果的，关键帧时间轴、缓动函数、入场出场顺序，随便一个做到位都要几十分钟。

Gamma、Beautiful.ai 这些 AI PPT 工具确实把制作时间压缩了。实测 Gamma 45秒就能生成一份完整演示文稿，Beautiful.ai 的智能排版也能自动调整布局。

但它们有几个硬伤：

**中文排版质量差。** 这些工具的核心用户群是英文用户，中文字体、行间距、段落节奏的调优几乎为零。长文科普内容放进去，出来的东西像把中文强行塞进了一个英文设计模板。

**全部是SaaS订阅制。** Gamma 免费版每月400积分（约10份演示），去水印要 $8/月；Beautiful.ai 没有免费版，Pro $45/月，团队版 $144/年/人。长期使用成本不低。

**输出是PPT，不是动画。** 它们生成的本质还是幻灯片，不是可以直接用于视频录制的动画文件。你要得到可播放的动画效果，还需要额外录屏。

**必须联网。** 所有计算都在云端，断网就罢工。

AI-Animation-Skill 走了完全不同的路线。

## 它做了什么不同的事

核心差异就一个： **输出物是单个HTML文件。**

HTML 文件意味着：浏览器直接打开就能看到完整的动画效果，不需要安装任何软件，不需要登录任何账号，不需要导出。你看到的就是最终效果。打开浏览器，按F11全屏，打开录屏软件，录制完成——从HTML到视频的转换只需要这一步。

技术上，44个模板全部是纯前端代码，零第三方框架依赖。最轻量的一个模板只有331行代码，最密集的包含30组VS对比动画——全部自己写的，没有调用任何外部库。

这个技术选择带来的直接结果：

- **本地运行，**
	不需要网络，不需要API密钥
- **零成本，**
	MIT开源协议，代码随便改
- **单文件输出**
	——一个HTML文件包含所有样式和逻辑，发给别人直接能看
- 跨平台，Chrome、Firefox、Safari、Edge 都能跑

## 44个模板到底有什么

44个模板分三类，每一类解决不同的场景需求。

### PPT Level2（26个）

这是项目最核心的资产。26个模板分成9个系列，每个系列针对特定类型的内容做了优化：

| 系列 | 数量 | 适用场景 | 亮点 |
| --- | --- | --- | --- |
| 系列1 | 1 | 概念引入、对比 | VS对比卡片 + SVG流程图 |
| 系列2 | 1 | 概念定义、层级结构 | 13种动画，最多元化 |
| 系列3 | 3 | 轻量/步骤/极简 | 3-3模板仅331行代码 |
| 系列4 | 3 | 案例/实验/代码 | 含代码雨动画效果 |
| 系列5 | 4 | 警示/失败/危险 | 5-4模板达15种动画+13页 |
| 系列6 | 4 | 护栏/架构/反馈 | 6-2红绿VS对比（15种动画） |
| 系列7 | 4 | 追踪/上下文/Doom Loop | 7-2模板达17种动画 |
| 系列8 | 3 | 辩论/对比/融合 | 8-3模板达30组VS对比 |
| 系列9 | 3 | 总结/共识/精炼 | 9-3模板仅5页，最精炼 |

科普视频常见的内容结构——概念解释、案例展示、问题分析、方案对比、总结提炼——这9个系列基本全覆盖了。

### PPT基础模板（4个）

4个通用基础模板，在Level2不适用时作为回退选项。其中 PPT-Generate-3 视觉效果最好，适合大多数通用场景。

### Animation流程图模板（14个）

这套模板把科普内容转化为平面UI风格的流程图动画。默认使用 RNN-3（分层卡片），其它模板针对特定技术概念做了深度优化：LSTM-1 专门展示LSTM的三阶段门控机制，word2vec-1 展示词向量的语义身份证概念，GPU 模板展示计算节点架构。

## 实际怎么用

前置条件：你需要一个 AI Agent 环境。AI-Animation-Skill 本身是提示词模板集，不是独立软件——它需要配合 AI Agent 来执行"理解文本 → 选择模板 → 生成HTML"这个过程。

目前支持的平台：

- **OpenClaw**
	（龙虾）——开源社区版
- **WorkBuddy**
	（腾讯版）
- **QClaw**
- **摩尔线程AIBOOK**
	（国产GPU平台）

### 安装

以 WorkBuddy 为例，三步完成：

**第一步：** 下载项目源码

```
git clone https://github.com/Unclecheng-li/AI-Animation-Skill.git
```

备注：也可以手动下载项目压缩包到本地

**第二步：** 复制到 skills 目录

将 `AI-Animation-Skill` 文件夹复制到 `~/.workbuddy/skills/` 目录下。

**第三步：** 重启 WorkBuddy

重启后，Agent 会自动识别这个 Skill。

如果你用的是 OpenClaw，可以通过 ClawHub 一键安装，不需要手动操作。

### 生成科普动画

安装完成后，使用流程极其简单。

**模式一：PPT演示生成**

在对话中输入你的科普内容，告诉 Agent "帮我生成PPT"。Agent 会自动完成三件事：

1. 分析内容类型，从26个Level2模板中选择最合适的一个
2. 生成基础HTML结构
3. 用选定模板重构，输出完整的动画HTML文件

你不需要告诉它用哪个模板，不需要手动调整参数，不需要做任何技术决策。

**模式二：流程图生成**

在模式一完成之后，如果你想把同样的内容转化为平面UI风格的流程图动画，只需要说一句"生成流程图"。Agent 会自动从14个Animation模板中选择合适的进行重构。

两种模式可以组合使用：先生成PPT版本用于完整展示，再生成流程图版本用于快速回顾。

### 从HTML到视频

生成的HTML文件直接用浏览器打开，全屏后用任何录屏工具录制即可。

OBS（免费）、Windows自带录屏（Win+G）、浏览器插件录屏，随便选一个。HTML的动画是确定性的——每次打开效果完全一致，不存在"录的时候动画没播好"的问题。

## 它的边界在哪

需要说清楚的不只是能做什么，还有不能做什么。

**不适合做品牌演示。** 模板风格偏向技术科普的简洁风，如果你需要企业级的品牌管控（锁定Logo、字体、配色），Beautiful.ai 的品牌套件做得更好。

**不适合做实时协作。** AI-Animation-Skill 是单人工作流，不支持多人同时编辑。Gamma 和 Beautiful.ai 都有实时协作功能。

**需要AI Agent环境。** 它不是一个独立软件，必须配合 OpenClaw、WorkBuddy 等 AI Agent 使用。如果你还没有这些环境，需要先搞定这一步。

**模板有限。** 44个模板对科普场景来说够用了，但如果你要做营销视频、产品宣传片、品牌广告，模板储备不够。

## 最后一件事

AI-Animation-Skill 做了一个取舍：用纯前端HTML替代PPT/After Effects作为动画输出格式，用AI Agent替代手动调参作为制作流程。

这个选择牺牲了通用性（只能做科普/教学类内容），换来了垂直场景下的极端效率（两步出成品）和零成本（开源免费、本地运行）。Gamma 和 Beautiful.ai 解决的是不同的问题，两者可以并存。但在"中文科普动画"这个具体场景下，AI-Animation-Skill 可能就是目前最好的方案。
