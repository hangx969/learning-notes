Server Migration Center：
- 支持多平台、多环境迁移：
	- 支持本地虚拟化服务器迁移
	- 操作系统迁移
	- VMWare无代理迁移
	- 支持跨账号、跨Region、跨AZ迁移ECS实例
	- 支持桌面迁移
- 迁移过程对源服务器无需停机
- 全量+增量迁移
- 迁移为ECS镜像、迁移到容器镜像、迁移到已存在的目标实例
- 断点续传

## 基础概念
- SMC客户端：会在源机器上安装一个agent，负责上报信息
- 中转实例：每次迁移的时候，会自动生成一台临时ECS实例，源服务器的数据会先传输到这上面，再由中转实例做镜像。
- 全量迁移：第一次迁移会全量迁移
- 增量迁移：每次增量迁移生成新的镜像，均为原服务器在当前时刻的全量数据镜像。
## 迁移时间评估
实际迁移时间 = 数据传输时间（源服务器传输 -- 中转服务器的数据传输速度，由源机器的出口带宽和中转服务器的入口带宽共同决定） + 镜像制作时间。

- 可以留有30% buffer
- 可以使用iPerf工具测试传输速度。
