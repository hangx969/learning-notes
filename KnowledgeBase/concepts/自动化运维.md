# 自动化运维

## 定义
自动化运维是通过工具和脚本替代手工操作，实现服务器配置管理、批量部署、状态巡检和故障自愈的运维模式。本仓库涵盖 Ansible 配置管理、Shell 脚本编写、Python 运维开发（fabric、paramiko）以及 SSH 远程管理等核心实践。

## 在本仓库中的位置
主要分布在 `Linux-Shell/` 目录（24 篇文章，涵盖 Linux 基础、Shell 脚本、SSH、Ansible）和 `Python/python-运维开发/` 目录（10 篇文章，涵盖 Python 运维脚本开发），两者结合构成完整的自动化运维知识体系。
