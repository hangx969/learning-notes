# 容器运行时

## 定义
容器运行时（Container Runtime）是负责在操作系统层面创建和运行容器的底层软件。Kubernetes 通过 CRI（Container Runtime Interface）与容器运行时交互。自 Kubernetes 1.24 起，containerd 成为默认运行时，取代了 Docker Engine 中的 dockershim。
