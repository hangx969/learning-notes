# MCP

## 定义
Model Context Protocol（模型上下文协议）是连接 AI 模型与外部工具、数据源的标准化桥梁协议。通过 MCP Server，AI 助手可以调用文件系统、数据库、API 等外部资源，实现工具增强的智能交互。

## 在本仓库中的位置
主要出现在 `AI/ClaudeCode/MCP.md`，是 Claude Code 生态的核心协议之一。也与 Obsidian 知识库搭建场景相关。
