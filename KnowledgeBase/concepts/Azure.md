# Azure

## 定义
Microsoft Azure 是微软提供的公有云计算平台，涵盖计算、存储、网络、容器、DevOps 等完整云服务体系。本仓库中以 Azure 为核心云平台，重点记录 VM、AKS、DevOps、Storage、Networking、Policy 等服务的实践笔记。
